# Workflow Management System

A backend application designed to manage workflows, tasks, users, approvals, and process tracking. The system provides REST APIs for creating workflows, assigning tasks, monitoring progress, and managing workflow states efficiently.

## 🚀 Features

- User Management
- Workflow Creation & Management
- Task Assignment
- Workflow Status Tracking
- Approval / Rejection Flow
- Role Based Access Control (RBAC)
- RESTful API Architecture
- Exception Handling
- Input Validation
- Database Persistence
- Layered Backend Architecture
